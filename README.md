# Machine-learning-basico-
Projeto de Machine Learning – IMDb Top 1000
Este projeto aplica técnicas de Machine Learning supervisionado sobre o dataset IMDb Top 1000, com o objetivo de construir e avaliar modelos preditivos a partir de dados reais do domínio de entretenimento.
O foco principal é demonstrar o pipeline completo de ML, desde a preparação dos dados até a avaliação dos modelos.
Objetivo:
Desenvolver um modelo capaz de prever uma variável-alvo relacionada aos filmes (como avaliação ou classe de desempenho), utilizando atributos numéricos e categóricos disponíveis no dataset.
Dataset:
O conjunto de dados contém informações como:
Ano de lançamento
Duração do filme
Nota do IMDb
Metascore
Receita (Gross)
Número de votos
Gêneros e outras variáveis descritivas
Etapas do Projeto:
Carregamento e inspeção dos dados
Limpeza e tratamento de valores ausentes
Codificação de variáveis categóricas
Normalização e padronização de atributos
Separação em conjuntos de treino e teste
Modelagem:
Treinamento de modelos de Machine Learning
Comparação entre diferentes algoritmos
Ajuste de hiperparâmetros quando aplicável
Validação dos resultados no conjunto de teste
Avaliação:
Uso de métricas apropriadas ao problema
Análise de desempenho e possíveis vieses
Interpretação dos resultados obtidos
Ferramentas Utilizadas:
Python
Pandas e NumPy
Scikit-learn
Matplotlib e Seaborn
Estrutura:
O projeto está organizado em um notebook Jupyter, com divisão clara entre pré-processamento, modelagem e avaliação, facilitando a leitura e reprodução.
Observações:
Este projeto tem caráter demonstrativo.
