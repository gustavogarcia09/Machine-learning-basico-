# Machine-learning-basico-
Projeto de Machine Learning para prever a nota de filmes do dataset IMDB Top 1000.
Problema tratado como regressão supervisionada, Modelos treinados: Regressão Linear e Random Forest
Avaliação feita com RMSE e R² em dados de teste, Random Forest apresentou melhor desempenho geral,
Projeto demonstra pipeline estruturado de ML, comparação de modelos e análise de resultados.
